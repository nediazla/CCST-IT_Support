# **Módulo 4: Problemas con el sistema operativo y las aplicaciones**

## 1. Resolución de problemas en sistemas operativos Windows

El técnico de soporte debe conocer las funciones, herramientas y configuraciones más comunes del sistema operativo **Windows** para asistir a los usuarios y mantener la estabilidad del sistema.

---

### 1.1 Configuración de pantalla y energía

**Problemas frecuentes:**
- Resolución incorrecta o pantallas duplicadas.
- Brillo no ajustable.
- Modo nocturno o ahorro de energía que apaga la pantalla.

**Soluciones:**
- Configuración → *Sistema → Pantalla* → ajustar resolución o detectar pantallas.
- Atajo rápido: `Win + P` → seleccionar modo de proyección.
- Ajustar brillo y energía: `powercfg.cpl` o “Opciones de energía”.
- Actualizar controlador de video en el *Administrador de dispositivos*.

---

### 1.2 Códigos BitLocker

**BitLocker** cifra unidades de disco para proteger los datos del usuario.

**Problemas comunes:**
- Solicitud de clave de recuperación tras un cambio de hardware o BIOS.
- Error al desbloquear la unidad.

**Solución:**
- Obtener la clave desde la cuenta Microsoft o administrador de dominio.
- Comando para verificar estado del cifrado:
```cmd
manage-bde -status
```
- Desbloquear unidad:
```cmd
manage-bde -unlock D: -RecoveryPassword CLAVE
```

---

### 1.3 Actualizaciones de Windows y aplicaciones

Mantener el sistema actualizado mejora la seguridad y el rendimiento.

**Verificación:**
- Configuración → *Windows Update*.
- Comando:
```cmd
wuauclt /detectnow
```
**Problemas comunes:**
- Fallo al instalar actualizaciones → ejecutar *Solucionador de problemas de Windows Update*.
- Espacio insuficiente → liberar disco con `cleanmgr`.

---

### 1.4 Memoria caché del navegador

**Problema:** el navegador no carga correctamente sitios o muestra datos antiguos.  
**Solución:**
- Borrar caché y cookies (Ctrl + Shift + Del).  
- Verificar extensiones problemáticas.  
- Probar en modo incógnito.  

**Navegadores:** Edge, Chrome, Firefox.

---

### 1.5 Finalización de procesos

**Administrador de tareas (Ctrl + Shift + Esc):**
- Pestaña *Procesos* → seleccionar tarea → *Finalizar tarea*.
- Permite ver consumo de CPU, RAM y disco.

**Comando alternativo:**
```cmd
taskkill /IM proceso.exe /F
```

---

### 1.6 Copias de seguridad y restauración en la nube

**Herramientas:** OneDrive, Google Drive, iCloud.  
Permiten sincronizar documentos y restaurar versiones anteriores.

**Pasos:**
1. Iniciar sesión en OneDrive.  
2. Seleccionar carpetas sincronizadas.  
3. Restaurar archivos desde *Historial de versiones*.

---

### 1.7 Inicio seguro y modo de recuperación

**Modo seguro:** inicia Windows con controladores mínimos.

**Acceso:**  
`Configuración → Recuperación → Inicio avanzado → Reiniciar ahora`  
→ *Solucionar problemas → Opciones avanzadas → Configuración de inicio → F4 (Modo seguro)*.

**Comando:**
```cmd
bcdedit /set {default} safeboot minimal
```

---

### 1.8 Funciones de accesibilidad

Windows incluye herramientas para mejorar la usabilidad:
- Lupa (`Win + +`).
- Narrador.
- Teclado en pantalla.
- Alto contraste.

**Ubicación:** Configuración → *Accesibilidad*.

---

## 2. Resolución de problemas en macOS

El técnico debe conocer herramientas equivalentes en **macOS** para resolver incidencias del sistema.

---

### 2.1 Configuración de pantalla y energía

**Soluciones:**
- Menú Apple → *Preferencias del sistema → Pantallas*.
- Detectar pantallas externas manualmente.
- Ajustar brillo o modo noche.
- Configuración → *Batería → Ahorro de energía*.

---

### 2.2 Permisos de aplicaciones

Cuando una app no accede a micrófono, cámara o disco:

**Ruta:** Preferencias → *Seguridad y privacidad → Privacidad*.  
Activar acceso a los recursos necesarios.

---

### 2.3 Montaje de unidades externas

**Verificación:**
- Finder → *Ubicaciones*.
- Comando Terminal:
```bash
diskutil list
diskutil mount /dev/disk2
```
Si no se monta, revisar formato (APFS, HFS+, exFAT).

---

### 2.4 Caché del navegador

**Solución general:**  
- Safari → Preferencias → Privacidad → “Administrar datos de sitios web”.  
- Chrome/Firefox → *Borrar datos de navegación*.  

---

### 2.5 AirDrop y transferencia de archivos

**Comprobaciones:**
- Activar Wi-Fi y Bluetooth.  
- Configurar “Solo contactos” o “Todos” en AirDrop.  
- Usar Finder → *AirDrop* → arrastrar archivos.

---

### 2.6 Finalización de procesos

**Monitor de actividad:** muestra CPU, memoria y procesos.  
- Forzar cierre: seleccionar proceso → *Forzar salida*.  
- Comando:
```bash
sudo killall nombre_proceso
```

---

### 2.7 Copias de seguridad y restauración

**Herramientas:** iCloud y Time Machine.  
Permiten restaurar archivos o todo el sistema desde una copia automática.

**Configuración:** Preferencias → *Time Machine → Seleccionar disco de copia de seguridad*.

---

### 2.8 Accesibilidad

Incluye lector de pantalla (VoiceOver), zoom, subtítulos, control por voz y combinaciones de teclado adaptadas.  
Configuración → *Accesibilidad*.

---

## 3. Dispositivos móviles

Los técnicos deben entender cómo diagnosticar problemas en **iOS** y **Android**.

---

### 3.1 Solución de problemas comunes

| Problema | Solución |
|-----------|-----------|
| No carga | Revisar cable, puerto y adaptador. |
| Sin señal | Reiniciar, modo avión o restablecer red. |
| Aplicaciones fallan | Borrar caché o reinstalar. |
| Correo no sincroniza | Verificar credenciales o servidor IMAP/SMTP. |
| Fallos de sistema | Reiniciar o actualizar software. |

---

### 3.2 Configuración de correo electrónico

**Datos necesarios:**
- Servidor entrante (IMAP o POP3).  
- Servidor saliente (SMTP).  
- Usuario y contraseña.  

**Ejemplo:**
- IMAP: `imap.office365.com`, puerto 993, SSL.  
- SMTP: `smtp.office365.com`, puerto 587, TLS.

---

### 3.3 Software de colaboración

Aplicaciones comunes:
- Microsoft Teams, Webex, Slack, Zoom, Google Meet.  
Solución: borrar caché, actualizar app o reinstalar.

---

### 3.4 Gestión mediante MDM (Mobile Device Management)

**MDM** permite a la empresa administrar dispositivos móviles:
- Configurar Wi-Fi, VPN, correo, políticas de seguridad.  
- Borrar remotamente en caso de pérdida.  
- Ejemplos: Cisco Meraki, Intune, Jamf.

---

## 4. Virtualización y nube

### 4.1 Conceptos básicos

| Término | Descripción |
|----------|-------------|
| **Máquina virtual (VM)** | Sistema operativo emulado dentro de otro. |
| **Hipervisor** | Software que permite ejecutar VMs. Ej: VMware, VirtualBox, Hyper-V. |
| **Snapshot** | Copia temporal del estado de una VM. |

**Ventajas:** aislamiento, pruebas seguras, reducción de costos.

---

### 4.2 Modelos de nube

| Modelo | Descripción | Ejemplo |
|--------|--------------|----------|
| **IaaS (Infraestructura como servicio)** | Servidores, almacenamiento y redes virtuales. | AWS EC2, Azure VM |
| **PaaS (Plataforma como servicio)** | Entorno para ejecutar aplicaciones. | Google App Engine |
| **SaaS (Software como servicio)** | Aplicaciones accesibles desde la web. | Microsoft 365, Gmail |

**Importante:** saber identificar el modelo correcto para escalar incidentes al equipo adecuado.

---

## 5. Resolución de problemas en aplicaciones

### 5.1 Instalación de aplicaciones

**Buenas prácticas:**
- Usar tiendas oficiales (Microsoft Store, Mac App Store, Play Store).  
- Evitar fuentes desconocidas.  
- Revisar permisos solicitados.  

---

### 5.2 Aplicaciones de productividad y colaboración

**Ejemplos:** Word, Excel, Outlook, Teams, Zoom.  
**Solución de fallos comunes:**
- Actualizar versión.  
- Reparar instalación desde *Panel de control → Programas → Reparar*.  
- Comprobar conexión de red o autenticación.

---

### 5.3 Errores frecuentes

| Problema | Causa | Solución |
|-----------|--------|-----------|
| No abre la aplicación | Archivos corruptos | Reinstalar o usar modo seguro |
| Error de licencia | Sesión expirada | Iniciar sesión nuevamente |
| Lentitud | Archivos temporales o caché | Limpiar caché |
| Fallo tras actualización | Incompatibilidad | Revertir versión o contactar soporte |

---

## 6. Ejemplo práctico

**Caso:** Outlook no sincroniza correos nuevos.  
**Diagnóstico:** revisar conexión, modo sin conexión, tamaño de archivo PST.  
**Solución:** desactivar modo sin conexión, reducir tamaño PST o crear nuevo perfil.  
**Resultado:** sincronización restaurada.  
**Documentación:** ticket cerrado con descripción y pasos aplicados.

## 7. Preguntas tipo examen

1. ¿Qué herramienta de Windows muestra el consumo de CPU y RAM?  
   A. PowerShell  
   B. Administrador de tareas  
   C. Visor de eventos  
   D. Panel de control  
   **Respuesta:** B  

2. ¿Cuál es el comando para verificar el estado de BitLocker?  
   A. bitlock /check  
   B. manage-bde -status  
   C. get-bitlockerstatus  
   D. cipher /status  
   **Respuesta:** B  

3. En macOS, ¿qué herramienta permite ver procesos activos?  
   A. Disk Utility  
   B. Monitor de actividad  
   C. Finder  
   D. Terminal  
   **Respuesta:** B  

4. ¿Qué significa MDM?  
   A. Manual Device Maintenance  
   B. Mobile Device Management  
   C. Managed Disk Module  
   D. Main Desktop Mode  
   **Respuesta:** B  

5. ¿Qué modelo de nube ofrece software completamente gestionado por el proveedor?  
   A. IaaS  
   B. PaaS  
   C. SaaS  
   D. HaaS  
   **Respuesta:** C  

6. ¿Qué comando en Windows inicia el modo seguro?  
   A. msconfig  
   B. safemode /boot  
   C. bcdedit /set {default} safeboot minimal  
   D. bootrec /rebuildbcd  
   **Respuesta:** C

   A. Instalarla con permisos de administrador.  
   B. Probarla en producción.  
   C. Bloquear su instalación y reportar la fuente.  
   D. Ignorar la advertencia del sistema.  
   **Respuesta:** C
